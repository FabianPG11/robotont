Praktikum 5¶

Enne praktikumi juurde asumist¶

  1. Mine lehele https://github.com/*sinu kasutajanimi***robotont-blank-scripts, kus asenda ***sinu kasutaja nimi oma kasutajanimega.
  2. Vajuta rohelist nuppu “Clone or download”
  3. Kopeeri sealt aadress
  4. Ava terminal ning trüki sinna cd catkin_ws/src
  5. Seejärel sisesta git clone -sinu-kopeeritud-link-
  6. Nüüd trüki terminali catkin build
  7. Kui see protsess on lõpetanud edukalt, sulge terminal ja oled valmis praktikumiga jätkama.

Enne praktikumi lõpetamist¶

  1. Ava uus terminal
  2. Sisesta sinna cd catkin_ws/src/robotont-blank-scripts
  3. Seejärel git add .
  4. Seejärel git commit -m “kommentaar mis praktikum oli”
  5. Seejärel git push
  6. Siin sisesta oma githubi kasutajanimi ja parool. Kontrolli, kas sinu github.com lehel uuenesid failid.
  7. Kui uuenesid, siis loo ssh-ühendus robotisse ning sisesta käsureale sudo poweroff.
